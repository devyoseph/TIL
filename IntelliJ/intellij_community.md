# IntelliJ Community

* 커뮤니티 버전을 사용해 Spring Boot 프로젝트를 진행한다.
* **주의점**: Spring 프로젝트 연결 전에 Julu11 등 사용할 JDK버전이 깔려있어야 한다.

## Build

* 위에서 다운받은 프로젝트의 압축을 푼다.
* IntelliJ를 열고 Open 항목을 이용해 압축해제한 프로젝트를 연결한다.
* Gradle 다운로드를 진행하면 완성되어있다.
